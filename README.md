## Installation
```bash
git clone git@github.com:ganguli-lab/proxalgs.git
cd proxalgs
pip install -r requirements.txt
python setup.py install
```

## Dependencies

Required:
- python 2.7 or higher
- numpy
- scipy
- toolz

Optional:
- scikit-image
- tableprint

## Development
